# Hardcore Achievements Plus

A Fabric mod that gives hardcore multiplayer a second chance — with a price. When a player dies, they stay dead as a spectator, and the mod assigns the living players a **revival achievement**. Complete it, and your friend comes back. Fail to progress, and the world slowly empties.

Revival achievements scale with your world's progression: early on you'll be asked to forge iron armor or survive the night; later, the world may demand a wither skeleton's head or the Ender Dragon itself.

## How it works

1. **A player dies.** Thunder rolls, a death card fills every screen, and the mod picks a revival achievement from the pool — weighted toward the *highest* tier your group has unlocked (80% top eligible tier, 20% one tier down).
2. **Everyone works on it.** The assignment is shared: any living player can complete it. Progress uses the vanilla advancement engine, so goals feel native — check the **Revival Achievements** tab in the advancements screen (L).
3. **Completion revives.** The fallen player returns at their own spawn point (world spawn as fallback) with full health and hunger, to a totem chime and a burst of particles.

If *everyone* dies, it's game over — until someone new joins and completes the standing revival achievement.

### Tiers

| Tier | Name | Stage |
|------|------|-------|
| 0 | Embers | Achievable bare-handed from nothing |
| 1 | Sparks | Iron age, first farms |
| 2 | Flames | Established base — diamonds, enchanting, villagers |
| 3 | Blaze | The Nether |
| 4 | Inferno | The End and late-game challenges |

Eligibility is gated by vanilla advancements: a tier-3 revival can only be assigned if at least one living player has genuinely reached that stage (e.g. holds `nether/obtain_blaze_rod`). Fifty achievements ship across the five tiers — most repeatable, a few one-time milestones (the Dragon, the Wither, a full beacon).

## Installation

- **Server**: required. Drop the jar in `mods/` alongside [Fabric API](https://modrinth.com/mod/fabric-api). The world must be **hardcore** (set `hardcore=true` *before* generating the world — the flag is baked into `level.dat` at creation).
- **Client**: *optional.* Vanilla clients get the full experience — chat announcements (hover an achievement name for its mission), title cards, sounds, and the native advancements tab. Installing the mod client-side additionally unlocks the revival panel on **H** (active revivals, available pool with unlock states, completion history).

Requires Minecraft 26.2, Fabric Loader ≥ 0.19.3, Java 25.

## Commands (OP)

```
/hap inventoryRestore <on|off>   Save inventories at death and return them on revival (default: off — items drop)
/hap status                      Show the toggle and all pending revivals
```

The restore decision is captured *at death time*, so flipping the toggle never retroactively changes a pending revival.

## Building from source

```sh
./gradlew build        # jar lands in build/libs/
./gradlew runClient    # dev client
./gradlew runServer    # dev server
```
